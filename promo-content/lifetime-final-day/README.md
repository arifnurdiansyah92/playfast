## Cara pakai

1. Setiap file punya **Subject A/B/C** — pilih satu, atau A/B test.
2. **Preheader** wajib dipasang (preview text di inbox).
3. Body sudah dalam format Markdown — paste ke editor email kamu (Brevo, Mailchimp, dll).
4. Gambar pakai Steam CDN — bisa langsung embed atau download dulu.
5. Link review **https://playfast.id/reviews** sudah ada di tiap email (anti-scam signal).
6. Sertakan ajakan **"forward ke teman"** di tiap email (bonus channel viral).
